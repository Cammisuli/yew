元素

标签结构

元素标签必须是自闭合的 <... />，或是每个标签都有一个对应的闭合标签。

{% tabs %} {% tab title="标签 - 闭合标签" %}

html! {
  <div id="my_div"></div>
}

{% endtab %}

{% tab title="无效" %}

html! {
  <div id="my_div"> // <- 缺少闭合标签
}

{% endtab %}

{% tab title="自闭合" %}

html! {
  <input id="my_input" />
}

{% endtab %}

{% tab title="无效" %}

html! {
  <input id="my_input"> // <- 没有自闭合
}

{% endtab %} {% endtabs %}

{% hint style="info" %} 为方便起见，一些 通常 需要闭合标签的元素是被允许自闭合的。例如，html! { <div class="placeholder" /> } 这样写是有效的。 {% endhint %}

Children

轻松创建复杂的嵌套 HTML 和 SVG 布局：

{% tabs %} {% tab title="HTML" %}

html! {
    <div>
        <div data-key="abc"></div>
        <div class="parent">
            <span class="child" value="anything"></span>
            <label for="first-name">{ "First Name" }</label>
            <input type="text" id="first-name" value="placeholder" />
            <input type="checkbox" checked=true />
            <textarea value="write a story" />
            <select name="status">
                <option selected=true disabled=false value="">{ "Selected" }</option>
                <option selected=false disabled=true value="">{ "Unselected" }</option>
            </select>
        </div>
    </div>
}

{% endtab %}

    <svg width="149" height="147" viewBox="0 0 149 147" fill="none" xmlns="http://www.w3.org/2000/svg">

Classes

有许多方便的选项可用于元素指定 classes：

{% tabs %} {% tab title="常量" %}

html! {
  <div class="container"></div>
}

{% endtab %}

{% tab title="多个属性" %}

html! {
  <div class="container center-align"></div>
}

{% endtab %}

{% tab title="插值" %}

html! {
  <div class=format!("{}-container", size)></div>
}

{% endtab %}

{% tab title="表达式" %}

html! {
  <div class=self.classes()></div>
}

{% endtab %}

{% tab title="元组" %}

html! {
  <div class=("class-1", "class-2")></div>
}

{% endtab %}

{% tab title="Vector" %}

html! {
  <div class=vec!["class-1", "class-2"]></div>
}

{% endtab %} {% endtabs %}

监听器

监听器属性需要传递一个由闭包包裹的 Callback。创建回调的方式取决于你希望你的应用程序如何响应监听器事件：

{% tabs %} {% tab title="Component 处理器" %}

struct MyComponent {
    link: ComponentLink<Self>,
}

enum Msg {
    Click,
}

impl Component for MyComponent {
    type Message = Msg;
    type Properties = ();

    fn create(_: Self::Properties, link: ComponentLink<Self>) -> Self {
        MyComponent { link }
    }

    fn update(&mut self, msg: Self::Message) -> ShouldRender {
        match msg {
            Msg::Click => {
                // 处理 Click
            }
        }
    }

    fn view(&self) -> Html {
        // 从组件 link 中创建回调来在组件中处理它
        let click_callback = self.link.callback(|_: ClickEvent| Msg::Click);
        html! {
            <button onclick=click_callback>
                { "Click me!" }
            </button>
        }
    }
}

{% endtab %}

{% tab title="Agent 处理器" %}

struct MyComponent {
    worker: Dispatcher<MyWorker>,
}

impl Component for MyComponent {
    type Message = ();
    type Properties = ();

    fn create(_: Self::Properties, _: ComponentLink<Self>) -> Self {
        MyComponent {
            worker: MyWorker::dispatcher()
        }
    }

    fn update(&mut self, _: Self::Message) -> ShouldRender {
        false
    }

    fn view(&self) -> Html {
        // 从 worker 中创建回调来在另一个上下文中处理它
        let click_callback = self.worker.callback(|_: ClickEvent| WorkerMsg::Process);
        html! {
            <button onclick=click_callback>
                { "Click me!" }
            </button>
        }
    }
}

{% endtab %}

{% tab title="其他情况" %}

struct MyComponent;

impl Component for MyComponent {
    type Message = ();
    type Properties = ();

    fn create(_: Self::Properties, _: ComponentLink<Self>) -> Self {
        MyComponent
    }

    fn update(&mut self, _: Self::Message) -> ShouldRender {
        false
    }

    fn view(&self) -> Html {
        // 创建一个短暂的回调
        let click_callback = Callback::from(|| {
            ConsoleService::new().log("clicked!");
        });

        html! {
            <button onclick=click_callback>
                { "Click me!" }
            </button>
        }
    }
}
